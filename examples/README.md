# 示例代码

本目录包含 GIS Code AI 的使用示例。

## 可用示例

### basic_usage.py
基本使用示例，展示如何：
- 创建 GIS 处理器
- 初始化 AI 模型
- 设置日志

运行示例：
```bash
python examples/basic_usage.py
```
